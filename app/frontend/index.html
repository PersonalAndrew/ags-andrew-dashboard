Estrutura do projeto

Base para crescimento

A plataforma foi pensada para evoluir sem misturar fontes de dados de forma desorganizada. Cada competição pode ter seu próprio módulo, enquanto os comparativos usam uma camada padronizada de métricas.

2022

Dados da Copa do Mundo 2022 com base no StatsBomb Open Data.

2026

Dados de partidas específicas da Copa 2026 com base em consultas ao SofaScore, começando pela análise de Inglaterra x Croácia.

Comparativo

Tabela padronizada para comparar seleções, jogadores e competições.
