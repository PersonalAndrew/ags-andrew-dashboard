Metodologia

Fonte e organização dos dados

Este módulo da Copa 2026 utiliza dados extraídos do SofaScore e migrados do dashboard original em Streamlit para uma página web integrada ao hub AGS Performance Análises. A estrutura foi separada em arquivos JSON para facilitar a evolução do projeto.

1

Os arquivos de estatísticas, finalizações, momentum, eventos, escalações e posições médias ficam organizados em assets/data/copa2026.

2

O JavaScript carrega os dados automaticamente e atualiza os cards e módulos visuais da página.

3

A próxima etapa será transformar os dados de chutes, jogadores, momentum e eventos em gráficos e visualizações interativas.
