# ⚽ Dashboard Copa do Mundo 2022

Dashboard interativo de análise de desempenho da **Copa do Mundo 2022**, desenvolvido com foco em **Análise de Desempenho** e **Ciência de Dados aplicadas ao futebol**.

O projeto transforma dados de partidas, jogadores e eventos em visualizações interativas, permitindo analisar jogos, seleções, rankings, finalizações, passes, posições médias, mapas de calor, momentos da partida, linha do tempo, ranking individual e resumo inteligente.

---

## 📌 Objetivo do projeto

O objetivo deste projeto é apresentar uma aplicação web voltada à análise de desempenho no futebol, utilizando dados da Copa do Mundo 2022 para criar um dashboard visual, interativo e organizado.

O projeto foi desenvolvido como parte da minha evolução em análise de dados esportivos, combinando programação, visualização de dados e interpretação tática.

---

## 🧠 Funcionalidades

- Resumo geral da competição
- Ranking por equipes
- Artilheiros
- Campanha do Brasil
- Lista de partidas
- Filtro por seleção
- Análise detalhada por partida
- Escalações
- Resumo inteligente
- Comparativo entre equipes
- Momentos da partida
- Linha do tempo dos eventos
- Ranking individual dos jogadores
- Posição média dos jogadores
- Mapa de calor
- Resumo estatístico dos jogadores
- Mapa de chutes
- Mapa de passes

## 📊 Dados

Os dados utilizados no projeto têm como base informações de partidas, eventos, jogadores, chutes, passes e escalações da Copa do Mundo 2022.

O banco de dados utilizado é SQLite, armazenado localmente no projeto.
